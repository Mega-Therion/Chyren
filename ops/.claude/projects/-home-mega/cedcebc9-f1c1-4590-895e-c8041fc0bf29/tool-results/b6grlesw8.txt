# Claude Skill Bundle

This folder mirrors the skill directories currently available in this workspace. It is organized so Claude Code's skill creator can inspect the real `SKILL.md` files, bundled references, scripts, and assets without relying on a hand-written summary.

## Layout

- `skills/codex/` - mirrored skills from `/home/mega/.codex/skills`
- `skills/agents/` - mirrored skills from `/home/mega/.agents/skills`
- `skills-manifest.json` - machine-readable inventory with descriptions and usage notes
- `INDEX.md` - human-readable catalog of every mirrored skill

## How To Use

1. Point the Claude Code skill creator at this folder.
2. Use `INDEX.md` to find the closest matching skill by trigger or description.
3. Open the mirrored `SKILL.md` for the exact workflow, references, and bundled resources.
4. When creating a new skill, copy the closest existing pattern and adapt the frontmatter, trigger language, and workflow sections.

## Inventory

- Codex skills mirrored: 83
- Agent skills mirrored: 35
- Total skills mirrored: 118

## Notes

- Duplicate or overlapping skill names are preserved by source tree, so you can compare variants instead of losing information.
- The bundle intentionally keeps the original structure intact so the skill creator can inspect real resources rather than a flattened approximation.
