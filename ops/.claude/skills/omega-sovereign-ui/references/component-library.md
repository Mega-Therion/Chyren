# Component Library

## Design Principles

- **Black Box Aesthetic**: High contrast, dark themes, minimal borders, subtle glows.
- **Glassmorphism**: Backdrop blur and slight transparency for layering.
- **Typography**: Clean, monospace or sans-serif (Inter/Fira Code).

## Core Primitives

- **Container**: `div` with `backdrop-filter: blur(10px)` and semi-transparent black background.
- **HUD Card**: Card with subtle glow/border highlights on state changes.
- **Interaction**: Subtle hover/active states; avoid noisy animations.
