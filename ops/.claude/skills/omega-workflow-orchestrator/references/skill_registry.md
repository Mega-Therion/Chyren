# Skill Registry

This document lists the available skills that the Omega Workflow Orchestrator can leverage.

## Core OmegA Skills
- **omega-web-tools**: Unified web search (Perplexity) and browser automation (Comet).
- **omega-sid-orchestrator**: Coordinate industrial reconnaissance (Perplexity, Comet, Neon Postgres).
- **skill-creator**: Create or update skills.
- **find-skills**: Discover and install new skills.
- **omega-workflow-orchestrator**: Orchestrate complex tasks (this skill).

## Priority Skills (Roadmap)
- **one-block-builder**: Generate exact, reproducible shell scripts.
- **repo-cartographer**: Map codebase structure and dependencies.
- **spec-to-code-auditor**: Verify implementation against documentation.
- **eval-forge**: Generate and run evaluation suites.

## Development Skills
- **backend-development**: General Node.js/TypeScript/Go backend tasks (REST APIs, DBs, Queues, Docker).
- **backend-dev-guidelines**: Specific guidelines for Langfuse-style backends (tRPC, Next.js, Prisma, ClickHouse).
- **neon-postgres**: Neon Serverless Postgres (Auth, Data API, CLI, SDKs).
- **fastapi-templates**: Production-ready FastAPI project setup.
- **fastapi-python**: Expert FastAPI development.
- **adk**: Botpress Agent Development Kit.

## Frontend Skills
- **senior-frontend**: Comprehensive modern frontend (React/Next.js/TS/Tailwind).
- **software-frontend**: Production-grade frontend (Next.js, Vue, Angular, Svelte).
- **vercel-react-best-practices**: Performance optimization for React/Next.js.
- **frontend**: VeChain dApp patterns (React Query, Chakra UI).
