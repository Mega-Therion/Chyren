# Skill Architect: The Meta-Skill
The `skill-architect` is a foundational skill used to bootstrap other highly autonomous, self-correcting agents within the OmegA ecosystem.

## Architectural Mandates
All skills generated by this tool must adhere to the OmegA-Architecture repository:
1. **AEGIS (Governance):** Enforce strict trust boundaries and safety constraints.
2. **AEON (Orchestration):** Time-indexed execution and state management.
3. **ADCCL (Control):** Adaptive dynamic command & control loops.
4. **MYELIN (Learning):** Continuous knowledge graph updates and memory refinement.

## Development Lifecycle
- **Constraint-First:** Every skill must define its limits and fallback error-handling strategies.
- **Audit-Traceable:** Every action must emit teleodynamic events for observability.
- **Self-Correcting:** Logic loops must include built-in validation steps.

## Workflow
1. Use `meta_generator.py` to create the initial skill structure.
2. Initialize `SKILL.md` using the OmegA-Architecture template.
3. Register using `npx skills add` to the local agent mesh.
