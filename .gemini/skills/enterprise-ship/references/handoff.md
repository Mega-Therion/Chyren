# Handoff Protocol

Final checklist for production handover:

- [ ] **Release Tag**: Git repository tagged with semantic versioning.
- [ ] **Release Notes**: Autogenerated from commit history since last tag.
- [ ] **Artifacts**: Signed images pushed to registry.
- [ ] **Notification**: Deployment status broadcast to all stakeholders.
- [ ] **Cleanup**: Ephemeral CI/CD environment resources purged.
