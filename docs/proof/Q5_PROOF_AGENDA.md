# Q5 Proof Agenda

## Goal

Resolve `Q5` from `docs/MASTER_EQUATION.md`:

> Is the holonomy group determined by commutators `[L_i, L_j]` of the effective Lindblad drift operators?

The target is not a rhetorical bridge. The target is a theorem candidate with explicit assumptions, objects, and failure conditions.

## Deliverable Shape

The work splits into two tracks and they must not be blurred:

- `Formal proof track`: definitions, lemmas, theorem candidate, proof or obstruction.
- `Executable witness track`: a toy model and numerical checks that instantiate the proof assumptions.

## Theorem Candidate

For a chosen constitutional manifold `M`, principal bundle `P -> M`, connection `chyren`, and a rigorously defined map from effective drift operators `L_k` to induced geometric fields `X_k`, the holonomy Lie algebra is generated by curvature values computed on those induced fields; therefore, under the stated assumptions, reachable holonomy classes are determined by the Lie algebra generated by drift commutators.

This statement is only meaningful if the objects below are explicit.

## Objects That Must Be Defined First

- `H`: ambient state space
- `M`: constitutional manifold
- `P -> M`: principal bundle
- `chyren`: connection 1-form
- `Chyren_curv = d chyren + chyren wedge chyren`: curvature 2-form
- `L_k`: effective drift operators
- `X_k`: induced tangent or horizontal fields
- `rho_t`: evolution law
- `state_to_M`: exact map from system state to manifold point

If any object remains metaphorical, the proof stops there.

## Proof Order

1. Fix the manifold model.
2. Define the connection explicitly.
3. Define the drift-to-geometry map `L_k -> X_k`.
4. Compute curvature on induced fields.
5. Relate `[X_i, X_j]` to `[L_i, L_j]`.
6. Invoke Ambrose-Singer correctly.
7. State the operational consequence for commuting and noncommuting drifts.

## Lemma Dependency Graph

- Lemma 1: the map from system state into `M` is well-posed.
- Lemma 2: the dynamics induce tangent evolution on `M`.
- Lemma 3: the chosen connection exists and is the one actually being used.
- Lemma 4: the drift operators induce a Lie-closed family of fields.
- Lemma 5: curvature on those fields is expressible via commutators.
- Lemma 6: the holonomy Lie algebra is generated by those curvature values.
- Theorem: drift commutators determine reachable holonomy classes under the stated assumptions.

## Toy Model

Build a minimal witness before attempting a grand proof:

- dimensions: start with `N = 3 or 4`, `m = 2`
- choose explicit matrices for `L_1`, `L_2`
- define an explicit path on `M`
- compute numerical parallel transport
- estimate curvature on small loops
- estimate holonomy around those loops

Required test cases:

- commuting drifts
- noncommuting drifts
- trivial holonomy case
- nontrivial holonomy case

If the toy model does not support the theorem candidate, revise the theory before scaling.
