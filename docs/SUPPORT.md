## 🔍 Getting Help

### 1. GitHub Issues

For **bugs** or **unexpected behavior**, open a GitHub Issue using the appropriate template:

- 🐛 [Bug Report](./.github/ISSUE_TEMPLATE/bug_report.md)
- ✨ [Feature Request](./.github/ISSUE_TEMPLATE/feature_request.md)
- 🔬 [ADCCL Drift Report](./.github/ISSUE_TEMPLATE/adccl_drift_report.md)

> ⚠️ **Do NOT open public issues for security vulnerabilities.** See [SECURITY.md](./SECURITY.md).

### 2. GitHub Discussions

For questions, ideas, and community interaction:

👉 [Open a Discussion](https://github.com/Mega-Therion/Chyren/discussions)

Use Discussions for:
- General questions about Chyren’s architecture
- Sharing ideas for new features
- Asking how something works
- Showcasing your use case
